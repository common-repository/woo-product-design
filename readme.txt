Woo-commerce product design.

== Description ==

Woocommerce product design allows to customize product before add to cart.

You must be installed Woocommerce plugin.

When you activated Woocommerce product design plugin then it will add new *Product Design* menu in admin.
Product customizer provide below functionality 
* Set product customizer base from admin.
* Set property of layer.
* Upload custom images.
* Add text.
* Preview design.
* Set Prise for customize.
* Many Other.

[Click Here](https://youtu.be/MWPvfjnVGbY) to view demo video. 
   
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `woocommerce-product-design` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress

After Activate the plugin you will found new metabox in product edit page and `Product Design` menu in admin.
